
# House Price Prediction Project

## Project Overview
This project predicts real estate prices in Bengaluru using various regression algorithms.

## Dataset
Bengaluru House Data - contains features like location, size, total_sqft, etc.

## Algorithms Used
- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost

## Evaluation Metrics
- R2 Score
- MAE
- RMSE

## Folder Structure
- `app.py`: Streamlit Application
- `models/`: Saved model and metadata
- `requirements.txt`: Dependencies

## How to Run Locally
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`

## How to Deploy
Push to GitHub and connect to Streamlit Community Cloud.
